News 30.05.2012

Jarvis 0.4 just arrived with the following features and fixes:
  • The update feed URL.
  • Gathering anonymous statistics.
  • Donate panel.
  • Update Preference panel.
  • Support for Xcode 4.3.
  • Incorrect display of the minutes in the hour if they where from 1 to 10.
  • The company name from buHuHu to Night Ideas Lab.
  • Added Contributors button and Contributors panel with contact buttons for me and the original developer.
  • Full-Screen Support.
  • Removed the "About Jarvis" title name from the About panel to be compliant with Mac OS X Human Interface Guidelines.
  • The ChangeLog menu now shows a local html page.
  • Preference panel.
  • Version and Build number in the About panel.
  • Support for 10.7 (Lion) and Xcode 4.2.1.
  • License panel and license to GNU GENERAL PUBLIC LICENSE version 3.
  • Minimum Operating System is 10.5 (Leopard).
