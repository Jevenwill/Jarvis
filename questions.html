Questions & Answers

What is Jarvis

This a morning default application. Currently it gathers and reads the following:

  • Greetings.
  • Time and date.
  • Calendar events.
  • Reminders (from iCal).
  • Weather conditions. (for Yahoo Weather)
  • Unread email count and VIP email count.
  • Location of tracked packages.
  • News headlines from NYTimes.
  • Daily quotations.

System Requirements

A Mac with Intel processor and with OS X 10.5 or better. If you have a PPC (Power PC) processor, OS X 10.4 or earlier or a Windows/Linux machine the application will not work.

Downloading

You can download the latest version of this project as an application from here.

Or you can download this project source code in either zip or tar formats.

Or you can also clone the project with Git by running: 

$ git clone git://github.com/NightIdeasLab/Jarvis

Installing

Just drag and drop the app into the /Application folder and your done.

Uninstalling

By removing the Jarvis application from the /Application folder.

Removing Jarvis won't remove all the files. So you need to remove the following files and folders :

- Folder com.NightIdeasLab.Jarvis
from
~/Library/Cache
- Folder com.NightIdeasLab.Jarvis.savedState
from
~/Library/Saved Application State
- File com.NightIdeasLab.Jarvis.plist
from
~/Library/Preferences
- File com.NightIdeasLab.Jarvis.plist.lockfile
from
~/Library/Preferences
- File com.NightIdeasLab.Jarvis.LSSharedFileList.plist
from
~/Library/Preferences
- File com.NightIdeasLab.Jarvis.LSSharedFileList.plist.lockfile
from
~/Library/Preferences
